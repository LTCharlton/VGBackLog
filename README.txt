# VGBackLog

-Motivation-
This program is for users with a finite allocation of time and money
These users will be able to optimise their gaming time



-Longterm-
Create program to help user cultivate and organizes game collection
Program will show user prices, estimated game time, and average review score
Recommendations will be generated for user to cultivate collection


-Shortterm-
Create platform to organize collection
Pull external information from API
User will be able to organize their collection based on time and money
